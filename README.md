# 贪吃蛇游戏（C++ EasyX 实现）
基于 C++ 和 EasyX 图形库开发的经典贪吃蛇小游戏，支持方向键控制、自动移动、吃食物变长、碰撞检测等核心功能，适配 Windows 平台（Visual Studio）。

## 📋 项目介绍
### 核心功能
- 方向控制：W/S/A/D 键控制蛇的上下左右移动
- 自动移动：无按键时蛇沿当前方向自动前进
- 食物系统：随机生成食物，吃到后蛇身变长、分数增加
- 碰撞检测：撞围墙/撞自身时游戏结束，显示最终分数
- 界面展示：网格背景、围墙、实时分数显示

### 技术栈
- 编程语言：C++（C++11 及以上）
- 图形库：EasyX Graphics Library（Windows 专属）
- 开发环境：Visual Studio 2022（推荐）

## 🛠️ 环境配置
### 前置条件
1. 操作系统：Windows 10/11（EasyX 仅支持 Windows）
2. 编译器：Visual Studio 2022（自带 MSVC 编译器）
3. EasyX 图形库：
   - 访问 [EasyX 官网](http://www.easyx.cn) 下载对应 VS2022 版本的安装包
   - 运行安装包，自动完成库文件配置

### 快速配置（Visual Studio）
1. 打开 Visual Studio 2022，新建「空项目」；
2. 将贪吃蛇核心代码文件（`snake.cpp`）添加到项目中；
3. 直接点击「本地 Windows 调试器」编译运行（无需额外配置）。

## 🎮 运行说明
### 操作指南
| 按键 | 功能 |
|------|------|
| W / ↑ | 蛇向上移动 |
| S / ↓ | 蛇向下移动 |
| A / ← | 蛇向左移动 |
| D / → | 蛇向右移动 |
| 无按键 | 蛇沿当前方向自动前进 |

### 资源文件（可选）
若需自定义图片背景/食物/围墙，需在项目根目录创建 `resources` 文件夹，放入以下文件：
- `300.jpg`：游戏背景图
- `snake.jpg`：侧边装饰图
- `围墙.jpg`：围墙纹理图
- `红苹果.jpg`：食物图片
> 若无需图片，代码已默认使用纯色图形替代，可直接运行。

### 运行效果
- 游戏窗口尺寸：1024×600
- 蛇移动速度：150ms/步（可修改代码中 `Sleep(150)` 调整）
- 分数规则：每吃一个食物分数 +10

## 📂 代码结构
snake.cpp
- ├── 全局变量：蛇长度、分数、蛇身坐标、方块宽度
- ├── 核心函数：
- │ ├── drawunit ()：绘制单个蛇身单元（圆形）
- │ ├── drawsnake ()：绘制整条蛇（区分身体 / 头部颜色）
- │ ├── bkuni ()：绘制围墙单元（图片 / 纯色矩形）
- │ ├── drawbk ()：绘制背景、网格、围墙、分数
- │ ├── snakemove ()：手动控制蛇的移动方向
- │ ├── automove ()：蛇的自动移动逻辑
- │ ├── foodcreat ()：随机生成食物（避免与蛇身重合）
- │ ├── foodshow ()：绘制食物
- │ ├── increase ()：检测吃食物，蛇身变长 + 分数增加
- │ ├── failjudge ()：碰撞检测（围墙 / 自身）
- │ ├── fail ()：游戏结束提示
- └── main ()：游戏主循环（初始化、事件监听、绘图刷新）

## 🐞 常见问题
### 1. 编译报错“找不到 graphics.h”
- 原因：未安装 EasyX 图形库，或安装版本与 VS 不匹配；
- 解决：重新下载对应 VS2022 版本的 EasyX，运行安装包修复。

### 2. 运行时图片无法加载
- 原因：`resources` 文件夹路径错误，或图片文件名不匹配；
- 解决：
  - 方式1：将 `resources` 文件夹放在项目根目录（与 `.sln` 文件同级）；
  - 方式2：改用代码中默认的纯色图形（无需图片）。

### 3. 游戏运行卡顿/闪烁
- 原因：未启用批量绘图；
- 解决：代码中已使用 `BeginBatchDraw()`/`FlushBatchDraw()`，无需额外修改。

### 4. Git 推送失败（连接 GitHub 超时）
- 原因：网络环境限制 HTTPS 连接；
- 解决：
  - 方案1：切换 SSH 协议（生成 SSH 密钥并添加到 GitHub）；
  - 方案2：配置 Git 代理（`git config --global http.proxy 代理地址`）；
  - 方案3：使用 GitHub Desktop 推送。
